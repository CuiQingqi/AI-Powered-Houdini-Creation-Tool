# AI-Powered-Houdini-Creation-Tool
AI-powered Houdini procedural asset creation tool. Chat in browser → AI plans modules/operations/nodetasks → auto-executes in Houdini. Supports DeepSeek/OpenAI/Claude/Ollama, Obsidian RAG, workflow save/load, viewport preview. Built with FastMCP + pure socket WebSocket bridge for Houdini 21 compatibility.
